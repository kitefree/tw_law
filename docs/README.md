# 前言

## 目標

翻寫既有台灣全國法規資料庫系統，使其系統能獨立運行於區域網路。最終達到嵌入題組內供考生查詢。

## 說明

1. 研發已完成，此系統可供入闈時使用。
2. 嵌入題組尚須與系統討論介接合作。

## 資源連結

1. [全國法規資料庫](https://law.moj.gov.tw/Index.aspx)

2. [政府資料開放平台](https://data.gov.tw/)
